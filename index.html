ALGORITHMS

A*

The A* algorithm is a method for finding the shortest path on a graph that uses a heuristic function to estimate the cost of the path to the goal, combining a greedy approach taking into account the distance already traveled.

You enter a training sample in csv text format, which is used to build a decision tree. Next, you can enter data in csv format to make a decision on the tree.
