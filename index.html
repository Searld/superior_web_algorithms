Algorithms.

AStar algorithm.

A* is a popular pathfinding and graph traversal algorithm that finds the shortest path between two nodes in a weighted graph.


It combines the strengths of Dijkstra's algorithm (guaranteed shortest path) and Greedy Best-First Search (fast exploration using heuristics)

Ant algorithm.

Ant Colony Optimization is a probabilistic algorithm inspired by the foraging behavior of ants.


It solves complex problems by simulating how ants lay down pheromone trails to find the shortest paths. Over time, stronger paths emerge as more ants follow and reinforce them.

Genetic algorithm.

Genetic Algorithm is an optimization method inspired by natural selection. It evolves a population of solutions through selection, crossover, and mutation.


Over successive generations, better solutions emerge by mimicking the process of biological evolution.

Clustering algorithm.

Clustering is an unsupervised machine learning technique used to group similar data points together. It identifies patterns or structures in data without prior labels.


The goal is to ensure items in the same cluster are more similar to each other than to those in other clusters.

Neuro.

A neural network for digit recognition learns to identify handwritten numbers by analyzing thousands of labeled examples.


It adjusts its internal parameters to minimize errors in prediction. Once trained, it can accurately classify new, unseen digits from images.

Tree of decision.

A decision tree algorithm learns from labeled data provided by the user in CSV format.


It builds a tree structure where each node represents a decision based on input features. New data is then passed through the tree to make predictions, with the path and outcome shown step by step.
